# بنك الفضة - المتجر الإلكتروني

متجر إلكتروني متكامل لبيع المجوهرات والساعات، يدعم اللغة العربية وتجربة مستخدم فاخرة.

## المميزات
- **متجر إلكتروني**: عرض المنتجات، سلة مشتريات، وإتمام الطلب.
- **تتبع الطلبات**: نظام تتبع حالة الطلب (قيد الانتظار -> جاري التحضير -> جاهز).
- **خيارات الاستلام**: توصيل للمنزل أو استلام من الفرع (مع خرائط).
- **لوحة تحكم**: للأدمن لإدارة الطلبات وتغيير الحالات.

## طريقة التشغيل

1. تثبيت الحزم:
```bash
npm install
```

2. تشغيل السيرفر:
```bash
npm run dev
```

3. فتح المتجر:
- المتجر: `http://localhost:3000`
- لوحة التحكم: `http://localhost:3000/admin`

## الصفحات الرئيسية
- `/` : الصفحة الرئيسية والمنتجات.
- `/cart` : سلة الشراء.
- `/track/[id]` : صفحة تتبع الطلب (تحصل على الرابط بعد الطلب).
- `/admin` : لوحة تحكم الإدارة.

## التقنيات المستخدمة
- Next.js 14
- Tailwind CSS / Vanilla CSS Variables
- Local JSON Database
